# Lc0 Engine Files

Файлы движка Lc0 не включены в Git репозиторий из-за ограничения размера файлов GitHub (100 МБ).

## Как получить файлы

### Вариант 1: Из релиза
Скачайте последний релиз с GitHub - все необходимые файлы будут включены в установщик.

### Вариант 2: Скачать вручную
1. Скачайте Lc0 с официального сайта: https://lczero.org/
2. Скопируйте следующие файлы в эту папку:
   - `lc0.exe`
   - `791556.pb.gz` (или другую версию весов)
   - `cublas64_11.dll`
   - `cublasLt64_11.dll`
   - `cudart64_110.dll`
   - `mimalloc-override.dll`
   - `mimalloc-redirect.dll`

## Требуемые файлы

- `lc0.exe` - исполняемый файл движка
- `791556.pb.gz` - файл весов нейронной сети
- `*.dll` - библиотеки CUDA (для GPU ускорения)
