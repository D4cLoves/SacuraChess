# Инструкция по созданию релиза

## Подготовка к релизу

1. **Обновите версию в проекте:**
   - Откройте `Installer/installer.iss`
   - Измените `#define MyAppVersion "X.X.X"` на новую версию

2. **Обновите README.md:**
   - Добавьте информацию о новых функциях
   - Обновите скриншоты (если есть)

3. **Соберите проект:**
   ```bash
   dotnet publish Lc-0_Chess/Lc-0_Chess.csproj -c Release -r win-x64 --self-contained false
   ```

4. **Создайте установщик:**
   - Откройте `Installer/installer.iss` в Inno Setup Compiler
   - Скомпилируйте установщик
   - Установщик будет в папке `Installer/Output/`

## Создание релиза на GitHub

### Автоматический способ (рекомендуется)

1. **Создайте и запушьте тег:**
   ```bash
   git tag v1.0.0
   git push origin v1.0.0
   ```

2. **GitHub Actions автоматически:**
   - Соберет проект
   - Создаст ZIP архив
   - Создаст релиз с прикрепленным архивом

### Ручной способ

1. Перейдите в раздел **Releases** на GitHub
2. Нажмите **"Draft a new release"**
3. Выберите тег (или создайте новый, например `v1.0.0`)
4. Заголовок: `Sakura Chess v1.0.0`
5. Описание:
   ```markdown
   ## Что нового
   - Описание изменений
   - Исправления багов
   - Новые функции
   
   ## Установка
   1. Скачайте `SakuraChessSetup.exe`
   2. Запустите установщик
   3. Следуйте инструкциям
   ```
6. Прикрепите файлы:
   - `Installer/Output/SakuraChessSetup.exe` (основной установщик)
   - `Installer/Output/SakuraChessSetup.7z` (если есть)
7. Нажмите **"Publish release"**
